HOW IT WORKS

End-to-End ML Pipeline

From raw order data to actionable delay predictions in seconds.

01
📥
Data Ingestion
Load and clean e-commerce order data. Drop PII fields, remove duplicates, and engineer the 3-class target variable.
02
⚙️
Feature Engineering
Create temporal features, market risk scores, region delay rates, shipping mode encoding, and discount impact metrics.
03
⚖️
Class Balancing
Apply SMOTE (Synthetic Minority Oversampling) to handle class imbalance, ensuring all three delay classes are well represented.
04
🌐
GNN + ML Models
Train CatBoost, XGBoost, LightGBM, Random Forest, and a Graph Neural Network that captures order-to-order relationships.
05
🏆
Model Selection
Evaluate all models on Weighted F1, Macro F1, Cohen's Kappa, and ROC-AUC. Automatically select and serialize the best performer.
06
🔮
Real-Time Prediction
Submit order details via the form. Get instant risk classification with confidence scores and actionable recommendations.
